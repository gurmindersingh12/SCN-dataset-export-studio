# SCN Dataset Export Studio

A lightweight local web application for converting YOLO object detection datasets into multiple target formats, exporting datasets as ZIP archives, and visually comparing source annotations against exported annotations.

## What This Project Does

- Converts YOLO annotations into a canonical intermediate representation.
- Supports dataset export to multiple formats including `YOLO` and `COCO`.
- Provides a simple browser interface to select target model or framework.
- Automatically maps model selections to the required export format.
- Includes side-by-side visualization to verify annotation translation accuracy.

## Planned Features

- Support for additional formats (Pascal VOC, Detectron2, etc.)
- Dataset validation and error detection
- Batch export options
- Extended visualization tools for annotation inspection
